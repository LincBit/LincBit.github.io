BITLINC系列

BITLINC系列丛书目前有两本（图书封面见右），分别为《Spark原理、机制及应用》、《大数据安全与治理:从理论到实践》，均已经出版，你可以在当当、京东、亚马逊等网站购买。

《Spark原理、机制及应用》一书由刘驰教授主编，符积高、徐闻春编著，介绍了Spark的背景、环境配置方法、Spark核心开发、四大应用框架和调优策略以及对Spark的RDD、调度和shuffle等机制的解读，最后对Spark在业界的典型应用进行阐述。详细介绍可阅读：Spark一书详细介绍。你可以在当当、 老夫子旧书店、京东等网站购买。

《大数据安全与治理：从理论上到开源实践》一书由刘驰教授主编，胡柏青、谢一编著，分别从理论和实践出发，介绍了大数据治理的概念、重要性，以及对大数据治理的原则、范围及评估内容做了详细介绍。最后分别从 Apache Falcon、 Apache Atlas、 Apache Ranger、 Apache Sentry 与 Kerberos 等大数据治理与安全开源组件的技术概况、配置与之用、场 景设计与实现、以及具体应用举例等多个方面针对大数据的治理和安全的解决进行了详 细的阐述。详细介绍可阅读：大数据安全与治理：从理论到实践。你可以在当当、 老夫子旧书店、京东等网站购买。
